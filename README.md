## Description
The challenge this week is to build the back end for an e-commerce site. You’ll take a working Express.js API and configure it to use Sequelize to interact with a MySQL database.

## Installation
    npm init
    npm install 
    npm install express
    npm install sequelize
    npm install mysql2
    npm install dotenv

## Usage
    mysql -u root -p
    npm run seed
    npm start
